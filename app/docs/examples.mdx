## Component Examples

### Button Component

Here's how a typical button component looks before and after obfuscation:

#### Before
```tsx
<button className="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded transition-colors duration-200 shadow-md hover:shadow-lg transform hover:scale-105">
  Click Me
</button>
```

#### After
```tsx
<button className="a1b2c3d4 e5f6g7h8 i9j0k1l2 m3n4o5p6 q7r8s9t0 u1v2w3x4 y5z6a7b8 c9d0e1f2 g3h4i5j6">
  Click Me
</button>
```

#### Rendered Result
<div className="flex gap-4 justify-center my-6">
  <button className="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded transition-colors duration-200 shadow-md hover:shadow-lg transform hover:scale-105">
    Before Obfuscation
  </button>
  <button className="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded transition-colors duration-200 shadow-md hover:shadow-lg transform hover:scale-105">
    After Obfuscation
  </button>
</div>

## Performance Impact

### Bundle Size Comparison

| Metric | Before Obfuscation | After Obfuscation | Improvement |
|--------|-------------------|-------------------|-------------|
| CSS File Size | 45.2 KB | 28.7 KB | 36.5% smaller |
| Class Name Length | 12.3 chars avg | 6.2 chars avg | 49.6% shorter |
| Total Classes | 1,247 | 1,247 | Same count |

### Loading Performance

The obfuscated CSS loads faster due to:
- **Smaller file size** - Less data to transfer
- **Shorter class names** - Faster CSS parsing
- **Reduced redundancy** - Optimized output

## Advanced Examples

### Responsive Design

Responsive classes are preserved and obfuscated:

```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4 p-4 sm:p-6 lg:p-8">
  <div className="bg-red-500 p-4 rounded">Item 1</div>
  <div className="bg-green-500 p-4 rounded">Item 2</div>
  <div className="bg-blue-500 p-4 rounded">Item 3</div>
</div>
```

### Dark Mode Support

Dark mode variants work seamlessly:

```tsx
<div className="bg-white dark:bg-gray-900 text-gray-900 dark:text-white p-6 rounded-lg border border-gray-200 dark:border-gray-700">
  <h2 className="text-2xl font-bold dark:text-white">Dark Mode Compatible</h2>
</div>
```
